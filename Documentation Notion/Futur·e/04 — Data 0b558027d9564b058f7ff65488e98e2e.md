# 04 — Data

La couche data de ***futur·e*** relie des sources publiques hétérogènes pour produire une lecture personnelle, locale et crédible du changement climatique et de ses effets.

Cette page documente trois choses :

- les sources publiques mobilisées ou envisagées
- le schéma logique de la base de données
- les règles de granularité géographique qui encadrent les croisements

Le but n’est pas d’accumuler des datasets. Le but est de savoir quelles données existent, à quelle échelle elles sont fiables, comment elles s’assemblent, et ce qu’elles permettent réellement de dire dans le produit.

### ***Exigences non négociables***

- traçabilité des sources
- respect de la granularité native
- séparation entre donnée brute, donnée normalisée et interprétation éditoriale
- prudence sur les inférences
- documentation continue des règles de croisement
